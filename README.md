# リポジトリ名：ストップウォッチ
選考課題のストップウォッチです。<br>
時間の計測開始（スタート）、計測一時停止（ストップ）、計測リセット（リセット）の三つの機能を含んでおります。<br>

## 使い方
●「スタート」押下で計測開始<br>
●計測中「ストップ」押下で計測停止<br>
●計測停止中に再度「スタート」押下で計測再開<br>
●計測停止中に「リセット」押下で計測時間を00:00:00にリセット<br>

## 注意
※計測中は「スタート」のみ押下可能<br>
※停止中は「スタート」「リセット」のみ押下可能<br>
※指定に基づき、1秒単位で計測して表示するようにしております。<br>
